## Why?

When AirPods are used as both input and output on a Mac, macOS switches to the SCO codec which dramatically reduces audio quality. Crisp fixes this by pinning the default input to your Mac's built-in microphone (or any input device you choose), so AirPods stay in high-quality AAC output mode.

## Install

1. Download the latest `.dmg` from the [releases page](https://github.com/Aayush9029/Crisp/releases/latest).
2. Open the `.dmg` and move **Crisp** to `Applications`.
3. Launch Crisp — it runs silently in your menu bar.
